## комбинаторика
основные задачи комбинаторики: подсчет числа элементов в конечных множествах с заданным условием. 
основные правила:
- правило умножения. пусть в $i$-том множестве $n_i$ элементов $(i = 1,\ ...,\ k)$. найдем $N$ - количество способов, которыми можно получить выборку $(a_1,\ ...,\ a_k)$, где $a_i$ из $i$-того множества. тогда $$N = n_1 \cdot\ ...\ \cdot n_k$$
- правило сложения. 

дописать

пусть имеется конечный набор элементов $a_1,\ ...,\ a_n$, из которого будут выбираться элементы. тогда это множество называется генеральной совокупностью. 

допустим из нее выбирается k элементов и располагаются в порядке их появления. это называется выборкой.
1. упорядоченные выборки (размещение). важно, в каком порядке мы доставали элементы из генеральной совокупности, т. е. выборка $(1,\ 2) \ne (2,\ 1)$
2. неупорядоченные выборки (сочетание). неважно, в каком порядке мы доставали элементы из генеральной совокупности, т. е. выборка $(1,\ 2) = (2,\ 1)$. важно только какие элементы были выбраны из генеральной совокупности. 

### случай, когда выбранный элемент возвращается в генеральную совокупность
это называется выбор с возвращением или с повторением.
физическая аналогия: выбор с возвращением - отсутствие запрета на нахождение нескольких частиц в одной ячейке.

1.1. **размещение с повторением**. посчитаем количество упорядоченных выборок с повторением. 
$$N = \bar{A}_n^k = n \cdot\ ...\ \cdot n = n^k$$
### случай, когда выбранный элемент не возвращается в генеральную совокупность
это называется выбор без возвращения или без повторения. 

1.2. **размещение без повторения**. посчитаем количество упорядоченных выборок без повторения. 
$$N = n \cdot (n-1) \cdot \ ...\ \cdot (n-k+1) = \dfrac{n!}{(n-k)!} = A_n^k$$
дописать

2.1. **сочетание с повторением**. посчитаем количество неупорядоченных выборок с повторением. поставим во взаимнооднозначное соответствие выборку из {0, 1}, где сначала встречается столько 1, сколько 1-ый элемент из генеральной совокупности встречается в выборке, потом 0, потом столько 1, сколько 2-ой элемент из генеральной совокупности встречается в выборке, ..., потом 0, потом столько 1, сколько n-ый элемент из генеральной совокупности встречается в выборке. тогда у нас будет множество выборок из {0, 1}, где n-1 раз встречается 0, а k раз встречается 1. тогда количество неупорядоченных выборок с повторением равно:
$$N = \bar{C}_n^k = C^k_{n+k-1}.$$
