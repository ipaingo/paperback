## 2. Как я выполняла работу
1. Определила микросервис - сервис уведомлений и коммуникаций
2. Описала таблицы: notifications, event_queue, templates
3. Продумала API
4. Описала форматы запросов и ответов
5. Описала POST /events в виде диаграммы активности

## 4. Как использовался ИИ
Я использовала ChatGPT для генерации идей. Модель предложила мне более подробный список функций микросервиса и список таблиц, после чего я выделила только ключевые на мой взгляд таблицы и описала их поля по-своему, как мне показалось лучше.
